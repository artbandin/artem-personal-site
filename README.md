# Artem personal site

Минимальный статический сайт Артёма Бандина.

## Локальный запуск

```bash
python3 -m http.server 8000
```

## Редактирование постов

Посты на главной странице берутся из `posts.md`.

Базовый формат:

```md
- [Название поста](https://example.com/post)
```

С описанием:

```md
- [Название поста](https://example.com/post) — короткое описание.
```

## Деплой

Сайт рассчитан на GitHub Pages или любой другой статический хостинг.
