Политика конфиденциальности расширения «Ответка»

Обновлено: 23 апреля 2026 г.

Этот документ описывает, какие данные обрабатывает расширение «Ответка» (далее — «расширение»), и что с ними происходит.

Коротко: расширение не собирает ваши данные. Своих серверов у расширения нет. API-ключ и настройки хранятся только локально в вашем браузере. Тексты отзывов уходят напрямую в OpenAI по вашему ключу.

Кто автор

Расширение сделано Артёмом Бандиным.

Какие данные обрабатывает расширение

Расширение обрабатывает три типа данных:

  1. Ваш API-ключ OpenAI. Вводите один раз в настройках расширения. Хранится локально, через стандартный API chrome.storage.local. Ключ передаётся только в OpenAI и только в заголовке Authorization запроса к их API. Никуда больше ключ не отправляется.
  2. Настройки расширения. Системный промпт, выбранная модель, temperature, max_tokens, лимит символов для контекста. Хранятся локально там же, где ключ.
  3. Тексты отзывов и (по вашему желанию) дополнительный контекст от бизнеса. Когда вы нажимаете «Придумать», расширение читает отзыв со страницы Яндекс.Бизнеса и отправляет его в OpenAI Chat Completions API вместе с вашим промптом. Ответ возвращается и подставляется в поле ответа. Дальше отправку ответа на Яндекс делаете вы вручную.

Что расширение не делает

  • Не отправляет никаких данных на серверы автора расширения — у расширения нет своих серверов.
  • Не использует аналитику, телеметрию, счётчики, cookie и трекеры.
  • Не читает другие страницы, кроме страниц отзывов Яндекс.Бизнеса, указанных в manifest.
  • Не хранит историю ваших отзывов, ответов или запросов.
  • Не делится данными с третьими лицами (кроме OpenAI, без которого расширение физически не работает).

Куда уходят данные

  • OpenAI — получает текст отзыва, имя автора, оценку, ваш системный промпт и опциональный контекст, когда вы нажимаете «Придумать». Обработка подчиняется политикам OpenAI: openai.com/policies/privacy-policy и openai.com/policies/api-data-usage-policies. По состоянию на апрель 2026 OpenAI не использует данные, отправленные через API, для обучения моделей без вашего явного согласия.

Никуда больше данные не уходят.

Как удалить данные

  • Удалите расширение через chrome://extensions/ — chrome.storage.local очищается автоматически вместе с расширением.
  • Или вручную: в настройках расширения сотрите API-ключ и нажмите «Сохранить».
  • Данные, уже отправленные в OpenAI, удаляются по правилам OpenAI (см. ссылки выше).

Разрешения расширения и зачем они нужны

  • storage — хранить API-ключ и настройки локально.
  • activeTab — отправлять команду генерации в текущую вкладку при нажатии горячей клавиши.
  • host_permissions на yandex.ru/sprav/* — встраивать кнопку «Придумать» на страницах отзывов Яндекс.Бизнеса.
  • host_permissions на api.openai.com/* — отправлять запросы к OpenAI API по вашему ключу.

Дети

Расширение не предназначено для детей и не собирает данные о детях.

Изменения политики

Если политика поменяется, актуальная версия будет лежать по тому же адресу и в репозитории. Дата в начале документа — дата последнего обновления.
